# MeterMeter

MeterMeter is a Neovim plugin for real-time poetic meter annotation.

Repository focus is now Neovim-first:
- plugin code: `nvim/metermeter.nvim/`
- smoke tests: `scripts/nvim_smoke_test.lua`
- test runner: `scripts/run_smoke_tests.sh`
- Python unit tests: `tests/test_nvim_*.py`

## Install

See `nvim/metermeter.nvim/README.md` for setup and full configuration.

## Development

Run full local checks:

```bash
./scripts/run_smoke_tests.sh
```
